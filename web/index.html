Supported Categories (10)

  • Browsers: Chrome, Firefox, Edge, Brave, Opera
  • Multimedia: VLC, OBS, Audacity, FFmpeg
  • Development: VS Code, Git, Node.js, Python, Docker
  • System: 7-Zip, WinRAR, Everything, CCleaner
  • Communication: Discord, Telegram, Slack
  • Office: LibreOffice, Notepad++
  • Security: Bitwarden, VPN tools
  • Gaming: Steam, Epic Games, GOG
  • Open Source: Blender, Krita, GIMP, Inkscape
  • Extra Tools: ShareX, Rufus, DBeaver, WireShark

System Requirements

  • Windows 10/11
  • PowerShell 5.1+
  • Administrator privileges
  • WinGet (Windows 10/11) or Chocolatey
